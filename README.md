WHAT AM I?

Eleven41.Logging is a general purpose logging framework, capable of logging to many different outputs.
Those include:
* Console
* File
* Windows Event Log
* Multiple outputs

Various logging levels exist, including:
* Diagnostic
* Info
* Warning
* Error
